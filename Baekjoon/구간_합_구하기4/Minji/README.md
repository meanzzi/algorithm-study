### 백준 11659 -  구간 합 구하기 4
---------------------------------------

## 1. 문제 설명
- 1번째 줄 : 수의 개수 N, 합을 구해야하는 횟수 M 주어짐
- 2번째 줄 : N개의 수가 주어짐 (배열 num)
- 3 ~ M번째 줄 : 구간 (i, j)가 주어진다.
	ex) 1 3이 주어지면 N[0]~N[2]의 합을 구해야함

## 2. 문제 해결
- 수의 개수와 합을 구해야 하는 횟수가 최대 100.000 이므로 구간마다 매번 계산하면 시간 초과가 발생할 듯
- N개의 수를 입력 받을 때 합 배열 만들기
```
ex) num = 12345 일때 S[0]=0. S[1]=1, S[2]=3, S[3]=6, ... , S[5]=15
	=> S[i] = S[i-1] + num[i]
```
- 구간 별로 합 구하기
```
ex) (2,3) 일때 2+3 = 5 이므로 (2,3)=S[3]-S[1]
	=> (i,j) = S[j] - S[i-1]
```
