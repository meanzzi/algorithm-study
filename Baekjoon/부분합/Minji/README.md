### 백준 1806 -  부분합
---------------------------------------

### 1. 문제 설명
- 연속된 수들의 부분합 중에 그 합이 s 이상 되는 것 중, 가장 짧은 것의 길이

### 2. 문제 해결
- 연속된 수들의 부분합 구해주기
- d[j] - d[i] >= s 이면 j++ 해주고, 최소 길이를 업데이트 해주기
- d[j] - d[i] < s이면, i++ 해주기
- 끝까지 다 탐색했는데도 결과가 없으면 0을 출력하고, 아니면 최소 길이 출력하기

문제 링크 : https://www.acmicpc.net/problem/1806
