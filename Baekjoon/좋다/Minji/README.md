### 백준 1253 -  좋다
---------------------------------------

### 1. 문제 설명
- n개의 수 중 **다른 수 두개**의 합으로 나타낼 수 있는 개수

### 2. 문제 해결
- n 범위 2000이므로 정렬 사용 가능
- 투 포인터를 사용해서 처음부터 끝까지 탐색하기
- start(v[i])와 end(v[j])의 값을 더했을 때 특정 값 (v[k])를 만족하는지 찾기
	- v[i] + v[j] < v[k] : i++
	- v[i] + v[j] > v[k] : j--
	- v[i] + v[j] == v[k] : count ++
		- 이때 (0+5)=5와 같은 상황을 만족하면 안됨 (문제에서 본인은 포함하면 안된다고 주어짐)
		- i!=k && j!=k 일때만 count ++ 해줘야함
