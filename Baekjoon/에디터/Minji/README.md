백준 1406 - 에디터

## 1. 문제 설명
- L : 커서를 왼쪽으로 한 칸
- D : 커서 오른쪽으로 한 칸
- B : 커서 왼쪽 문자 삭제
- P $ : $라는 문자를 커서 왼쪽에 추가

## 2. 문제 풀이
- 두 개의 스택을 만들어서 왔다갔다 함. 한 개는 입력받은 스택의 복사본
- 왼쪽으로 한 칸 옮길때는 스택의 top을 복사하고 pop함
- 오른쪽으로 한 칸 옮길 때는 복사본의 top을 가져옴
- 삭제는 pop, 추가는 push함
- 모든 작업 완료 후 기존 스택을 복사본에 다 옮기고 복사본을 출력함
