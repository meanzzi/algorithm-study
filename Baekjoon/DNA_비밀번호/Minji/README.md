### 백준 12891 - DNA 비밀번호
---------------------------------------

### 1. 문제 설명
- DNA 문자열에 등장하는 문자 : A, C, G, T
- 임의의 부분 문자열을 뽑았을 때 ACGT의 최소 개수를 만족해야 비밀번호로 사용 가능

### 2. 문제 해결
- 입력 값이 1,000,000 크므로 슬라이딩 윈도우 사용
```
CCTGGATTG 
```
A C G T
2 0 1 1
- 위의 조건을 최소로 만족해야하고 임의로 뽑을 부분문자열 길이가 8(S)이므로
```
CCTGGATT
```
A C G T
1 2 2 3 
- A가 조건을 만족하지 못했다.
```
CTGGATTG
```
A C G T
1 1 3 3
- 첫 문자(C) 개수 -1, 마지막 문자(G) 개수 +1 또한 최소 조건 만족 못하므로 결과는 0

=> 새로 들어오는 문자와 제거되는 문자의 개수만 업데이트 하여 개수 확인하기
