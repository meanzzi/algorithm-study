### 백준 1377 - 버블 소트
---------------------------------------

### 1. 문제 설명
버블 정렬을 더 이상 하지 않아도 될 때 횟수는? (문제에 주어진 코드를 해석해야함)

### 2. 문제 해결
- 10 1 5 2 3 이라는 입력 값이 주어졌을 때
```
정렬
10 1 5 2 3 (입력 값)
1 2 3 5 10 (정렬 후)

index
1 3 4 2 0 (정렬 전)
0 1 2 3 4 (정렬 후)
------------
1 2 2 -1 -4 (정렬 전 - 정렬 후)
```
따라서, max의 값이 2이다. 
답은 max+1을 해주면 된다. 

(마지막에 반복문을 한 번 더 돌리고 swap을 하지 않아도 된다고 판단하기 때문이다. -> 문제 조건 코드 확인)
