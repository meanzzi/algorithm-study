### 백준 10986 -  나머지 합
---------------------------------------

## 1. 문제 설명
- n개의 수가 주어지고 그 수들의 부분합이 m으로 나누어 떨어지는 구간의 개수

## 2. 문제 해결
- 시간 초과 나지않기 위해 누적합으로 계산하기
- 누적합의 배열에 m으로 나누어떨어져야 하므로 각 배열 인덱스마다 %m을 해주기
```
ex) num = 12312 일때 누적합 배열 S = 13679
	=> %m을 하면 10010
```
- 0일때는 나누어 떨어지므로 count++
- 같은 수인 경우의 수 구하고 count에 더해주기
```
ex) S[4]-S[0] % m == 0 이므로 나누어떨어짐 (0~4의 구간을 구하는 것임)
```
