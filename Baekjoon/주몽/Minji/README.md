### 백준 1940 - 주몽
---------------------------------------

### 1. 문제 설명
- 두 재료의 고유한 번호를 합쳐서 (M) 갑옷을 만들 때, 몇 개를 만들 수 있는지 구하기

### 2. 문제 해결
- n의 범위가 15000인데 2초이므로 정렬을 해도 시간초과가 되지 않으므로, 정렬을 해준다.
- start(v[i]), end(v[j])라고 할 때
- v[i] + v[j] < m : i++
- v[i] + v[j] > m : j--
- v[i] + v[j] == m : count++; i++, j--;

문제 링크 : https://www.acmicpc.net/problem/1940
