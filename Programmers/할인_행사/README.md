# 풀이 과정
### 1. 문제 정리
- 사고싶은 물건과 행사하는 물건 배열이 주어졌을 때, 언제부터 물건을 사야 모든 물건을 할인행사 가격에 살 수 있는지 구해라.
- 살 물건의 개수는 10개이다.
- 살 수 있는 모든 날짜의 개수를 반환한다.

### 2. 풀이 과정
- 사야하는 물건의 개수를 오브젝트에 저장한다.
- 할인 행사하는 배열을 순회하며 10일 뒤까지 모두 구매했을 때, 사고싶은 물건을 모두 샀는지 체크한다.
